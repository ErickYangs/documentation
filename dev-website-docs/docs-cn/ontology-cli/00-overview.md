本体客户端 Ontology CLI 是一个用 Go 语言编写的命令行应用程序，其主要功能如下：

- 运行和管理全节点
- 管理钱包帐户
- 管理原生资产
- 管理区块数据
- 显示区块链信息
- 部署和调用智能合约
- 构建、签署和发送交易
